# Генератор и Проверка Паролей

Программа с графическим интерфейсом для генерации, проверки сложности и безопасного хранения паролей.

## Возможности программы

- Генерация паролей с настройкой длины и типов символов
- Проверка сложности паролей с расчётом энтропии
- Хранение паролей в локальной базе данных
- Контроль целостности данных с помощью SHA-256 хэшей
- Копирование паролей и логинов в буфер обмена

## Как запустить

### Способ 1: Готовый исполняемый файл (рекомендуется)

1. Перейдите в раздел **Releases** этого репозитория
2. Скачайте файл для вашей операционной системы:
   - `password_app.exe` — для Windows
   - `password_app-linux` — для Linux
3. Запустите скачанный файл

### Способ 2: Запуск из исходного кода

1. Убедитесь, что у вас установлен Python версии 3.6 или выше
2. Скачайте исходный код
3. В терминале перейдите в папку с проектом
4. Запустите программу командой: `python password_app.py`

## Примечания

- При первом запуске программа создаст файлы `passwords.json` и `passwords.hash`
- Все пароли хранятся локально на вашем компьютере
